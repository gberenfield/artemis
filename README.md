Artemis
=======
Nifty prompt for searching strings.

Features
========
* Small/fast/low-dependency (ncurses)
* Strings from flat-file or stdin
* Dumps choice to stdout or file
* tab-completion of strings/searching

Building
========
gcc main.c -ncurses -o artemis
